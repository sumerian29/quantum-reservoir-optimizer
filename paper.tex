\title{Reinforcement Learning-Guided Quantum Optimization for Physics-Informed Well Placement Under Uncertainty}

\author{Tareq Mageed}
\address{Ministry of Oil, Iraq}

\begin{abstract}
This study presents a next-generation hybrid quantum--classical framework for optimal well placement that integrates physics-informed QUBO modeling, advanced QAOA, reinforcement learning, and uncertainty quantification. Unlike conventional approaches, the proposed method uses reinforcement learning to adaptively optimize QAOA parameters within a quantum-in-the-loop framework. The method is validated using a Volve-based dataset and surrogate reservoir simulations. Results demonstrate that the proposed approach achieves an approximation ratio of 0.98, while outperforming classical optimal solutions in net present value (NPV) by up to 40\%. The framework further demonstrates robustness under geological uncertainty and provides a scalable pathway toward real-field quantum optimization.
\end{abstract}

\section{Introduction}

Well placement optimization is a combinatorial NP-hard problem. Classical methods suffer from scalability and local minima issues. Quantum-inspired methods such as QAOA provide new opportunities but require further development for real-world applications.

This work introduces:
\begin{itemize}
\item Physics-informed QUBO formulation
\item Reinforcement learning guided QAOA
\item Uncertainty-aware optimization
\item Quantum-in-the-loop simulation framework
\end{itemize}

\section{Methodology}

\subsection{Physics-Informed QUBO}

\[
E(x) = x^T Q x - \alpha \sum x_i - \beta \sum_{i<j} x_i x_j + \gamma \sum_{i<j} \frac{P_i P_j}{d_{ij}} x_i x_j
\]

\subsection{Constraint Encoding}

\[
E_{constraint} = \delta (\sum x_i - k_{target})^2
\]

\subsection{Uncertainty Modeling}

\[
P_i \sim \mathcal{N}(\mu_i, \sigma_i^2)
\]

Expected objective:

\[
\mathbb{E}[E(x)] = \int E(x, \theta) p(\theta) d\theta
\]

\subsection{QAOA Formulation}

\[
|\gamma,\beta\rangle = \prod_{l=1}^{p} e^{-i\beta_l H_M} e^{-i\gamma_l H_C} |+\rangle^{\otimes n}
\]

Custom mixer:

\[
H_M = \sum_i X_i + \eta \sum_{i<j} I_{ij} X_i X_j
\]

\subsection{Reinforcement Learning Integration}

We model QAOA parameter tuning as a Markov Decision Process:

\begin{itemize}
\item State: $(\gamma, \beta)$
\item Action: parameter update
\item Reward: NPV from simulation
\end{itemize}

Policy is optimized using PPO.

\subsection{Quantum-in-the-loop Optimization}

Workflow:

\begin{enumerate}
\item Generate QAOA candidate solutions
\item Evaluate using reservoir simulator
\item Update RL policy
\item Iterate
\end{enumerate}
